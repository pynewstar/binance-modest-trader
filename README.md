# 简介

收益图是币安的手动实盘，一时兴起的100U账号，其实翻多少倍不是难点，控制回撤率才是精髓。此后花了很多时间思考关于收益和回撤顾此失彼的一些改良方案和手动的盘感量化问题，着力于对手动逻辑中的每一个特征点模块化，以便维护。
<br><br>
从策略设计来说，跟手动操作的分析模式略有区别，但都是着重于对仓位的管理设计，所以在回撤控制的特性手动实盘与回测曲线比较相似。
